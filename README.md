# Interactive Periodic Table

An interactive web-based periodic table built using **HTML, CSS, and JavaScript**.
Users can click on any element to view detailed information such as atomic number, category, atomic mass, and a short summary.

---

## 📌 Features

* Displays the **complete periodic table (118 elements)**.
* Interactive element cards.
* Click an element to view:

  * Atomic Number
  * Symbol
  * Name
  * Atomic Mass
  * Category
  * Element Summary
* Responsive layout that works on **desktop, tablet, and mobile**.
* Clean and minimal UI design.

---

## 🛠 Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and responsive layout
* **JavaScript (Vanilla JS)** – Dynamic rendering and interactivity

## 🎯 How It Works

1. The periodic table layout is created using **CSS Grid**.
2. Element data is stored in a **JavaScript array of objects**.
3. JavaScript dynamically generates each element cell.
4. Clicking an element triggers a **details panel** displaying its information.

---

## 📱 Responsive Design

The table automatically adjusts cell size based on screen width using CSS media queries:

* Desktop view
* Tablet view
* Mobile view
